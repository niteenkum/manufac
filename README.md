## Run Locally

Clone the project

```bash
  git clone https://github.com/niteenkum/manufac
```

Go to the project directory

```bash
  cd manufac
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start

  View on http://localhost:3000/
```
